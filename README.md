
E-Commerce Website Frontend

A responsive and modern E-Commerce Website Frontend developed using HTML, CSS, and JavaScript. This project provides a clean and user-friendly shopping experience with a well-organized interface and responsive design.

Features

- 🔐 Login & Signup Authentication UI
- 🏠 Home Page
- 🛍️ Product Listing Page
- ✨ New Collection Section
- 👩 Women's Collection
- 👶 Kids Collection
- 🛒 Shopping Cart Page
- 💬 Message/Contact Page
- 📱 Fully Responsive Design
- 🎨 Clean and Modern User Interface

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

Project Purpose

This project was built to strengthen frontend web development skills by creating a complete e-commerce user interface using only HTML, CSS, and JavaScript. It focuses on responsive layouts, clean code, and an intuitive user experience.
